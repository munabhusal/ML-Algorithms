# From Scratch to Production: Machine Learning Algorithms

A collection of machine learning algorithms implemented from scratch using **NumPy** and **Pandas** to understand the mathematics behind classical machine learning models. Each implementation is built from first principles, validated against **Scikit-Learn**, and focuses on both theory and optimization.

### Implemented Algorithms

* Decision Tree (Entropy, Gini Impurity, Information Gain)
* Linear Regression (OLS & Batch Gradient Descent)
* Logistic Regression (Gradient Descent)
* Gradient Descent (Batch, Stochastic & Mini-Batch)

### Highlights

* Scratch implementations using only NumPy and Pandas
* Vectorized computations for improved efficiency
* Side-by-side comparison with Scikit-Learn
* Focus on mathematical intuition, optimization, and performance analysis

### Tech Stack

* Python
* NumPy
* Pandas
* Matplotlib
* Plotly
* Scikit-Learn (for benchmarking)

### Goal

The aim of this repository is to bridge the gap between machine learning theory and implementation by building core algorithms from scratch instead of relying solely on high-level libraries.
